# ✂️ Corte Fino — Barbershop Landing Page

Landing page para barbería ficticia desarrollada como proyecto de portafolio.

## 🛠️ Tecnologías
- HTML5
- CSS3 (variables, grid, flexbox, animaciones)
- JavaScript Vanilla

## ✨ Características
- Diseño responsive (móvil y escritorio)
- Navbar fijo con efecto scroll
- Animaciones al hacer scroll
- Menú hamburguesa para móvil
- Integración con WhatsApp
- Galería con grid asimétrico
